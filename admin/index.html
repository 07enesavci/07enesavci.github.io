Giriş

Parolan hiçbir yere kaydedilmez. GitHub token'ı bu parolayla şifrelenmiş olarak yalnızca bu tarayıcıda saklanır; doğru parola olmadan çözülemez. "Sıfırla" kayıtlı token'ı siler ve ilk kuruluma döner.

İlk Kurulum

Token oluşturmak için: GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens → "Generate new token". Repository access'i yalnızca bu repo ile sınırla, Permissions altında Contents: Read and write ver. Belirlediğin parola token'ı bu tarayıcıda şifreler; parola hiçbir yerde saklanmaz, kaybedersen token'ı yeniden girmen gerekir.
